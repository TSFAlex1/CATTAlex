How to set up Beat Saber mods using BSManager (PC)

This guide walks you through installing and managing Beat Saber mods on PC using BSManager. Read each step carefully and make backups before modifying game files.

  1. Before you start — prerequisites
    • Beat Saber installed on PC (Steam or Microsoft Store)
    • Windows user with admin rights (recommended for installs)
    • BSManager (download from the project's official releases page)
    • Optional: make a quick backup of your Beat Saber folder (just copy the install folder to a safe place)
  2. Download & run BSManager
    1. Get BSManager from the official repository/releases (always use the project's official page).
    2. Extract the zip (if needed) and run the BSManager executable.
    3. If Windows warns about unknown publisher, confirm you trust the source (only if you downloaded from the official page).
  3. Point BSManager to your Beat Saber installation

    When first opened, BSManager will ask where Beat Saber is installed. Choose the folder that contains Beat Saber (for Steam this is usually under your SteamLibrary/steamapps/common/Beat Saber).

  4. Install the mod loader and core dependencies
    • BSManager usually detects and installs required plugin loaders (BepInEx/IPA or similar). Use the UI option to install the loader if prompted.
    • Install the core dependencies first — many mods require these to function.
  5. Browse and install mods
    1. In BSManager you'll see a list of available mods or an install dialog. Search or select mods you want (e.g., NoodleExtensions, Chroma, MappingExtensions, Camera2, CustomSabersLite, Vivify, ReeSabers, BeatLeader, ScoreSaber, BeatSaberPlus).
    2. Click Install (or Apply). BSManager will download and place the mod files in the correct locations.
  6. Enable and configure mods
    • After installation, check the mods list — ensure mods are enabled.
    • Some mods have configuration files or in-game menus. Open Beat Saber and look for mod-specific settings (or edit config files under the BepInEx/config folder if needed).
  7. Start Beat Saber and verify
    1. Launch Beat Saber from Steam (or the store) and confirm the game starts without errors.
    2. Open the mods menu or check for mod features to verify each mod loaded correctly.
  8. Updating mods
    • Use BSManager's update feature to keep mods and the loader up to date.
    • Before updating many mods at once, make a quick backup in case an update causes issues.
  9. Troubleshooting
    • Game won't start / crash on launch: Temporarily remove recently installed mods, then add them back one-by-one to find the culprit.
    • Black screen or missing visuals: Make sure BSManager installed the correct loader and that OBS/other programs are not interfering. Run Beat Saber before selecting it in capture tools.
    • Antivirus/quarantine: Some mod files can be flagged. Check your security software and allow/whitelist the Beat Saber folder if you trust the files.
    • Conflicting mods: Some mods may overlap functionality — check mod pages for compatibility notes.
  10. Best practices & tips
    • Keep a small "test" set of core mods working before adding lots of extras — that makes troubleshooting far easier.
    • Read each mod's install/compatibility notes on its official page or GitHub.
    • Use BSManager to uninstall or roll back mods rather than manually deleting files when possible.
    • When an update breaks things, revert to your backup and update mods one-at-a-time.
  11. Uninstalling / cleaning up
    • Use BSManager's uninstall/remove features where available.
    • If you need a full clean reinstall, restore your backed-up folder or reinstall Beat Saber from Steam.
