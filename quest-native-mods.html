Quest-native Beat Saber mods (BMBF / SideQuest) — step-by-step

This guide covers the commonly used flow for installing mods and custom songs on Oculus/Meta Quest devices using SideQuest and BMBF. Read everything first and make backups where suggested. Modding native Quest Beat Saber modifies the game on-device and carries some risk — proceed carefully.

Important warnings

  • Modding can break with official updates — automatic updates may unpatch Beat Saber. Disable auto-updates for Beat Saber in the Oculus app if you want to keep mods.
  • Follow official project pages (SideQuest, BMBF) — avoid shady downloads.
  • Always backup your Beat Saber data (and the entire headset if possible) before patching.
  • Using mods may violate the game's terms of service — do this at your own risk.

Prerequisites

  1. Meta Quest or Quest 2 with a working USB cable (or use SideQuest wireless after setup)
  2. A PC (Windows/macOS/Linux) — SideQuest runs on all platforms
  3. Beat Saber installed on the Quest (the version should be compatible with the mod tool you use)
  4. An Oculus/Meta account and the mobile Oculus app to enable Developer Mode on the headset
  5. Patience — some steps require confirmations inside the headset

Overview (what you'll do)

  1. Enable Developer Mode for your Quest via the Oculus mobile app
  2. Install SideQuest on your PC and connect the headset
  3. Use SideQuest to install BMBF (or the current Quest modding tool)
  4. Open BMBF in VR and follow its patching flow — this patches Beat Saber so custom songs & mods work
  5. Use BMBF to install custom songs and compatible Quest-native mods

Step-by-step instructions

1) Enable Developer Mode on your Quest

  1. Open the Oculus (Meta) mobile app on your phone.
  2. Select your headset, go to Settings → More Settings → Developer Mode and toggle it on.
  3. If you don't have an organization, you may need to create a free developer organization — follow the app prompts.

2) Install SideQuest on your PC

  1. Download SideQuest from https://sidequestvr.com/ and install it on your PC.
  2. Run SideQuest and connect your Quest via USB. Approve the Allow USB debugging prompt inside the headset when it appears (tick "Always allow" if you plan to use it regularly).
  3. Confirm SideQuest shows a green dot (connected).

3) Install BMBF (mod manager) via SideQuest

  1. In SideQuest, search for BMBF in the app list or use the Install APK option with the official BMBF build. Official BMBF info: https://bmbf.dev/ (check for latest method and download links).
  2. Install BMBF to the headset with SideQuest. Wait for installation to finish.

4) Run BMBF and patch Beat Saber (in VR)

  1. Put on your Quest, open Apps → Unknown Sources → open BMBF.
  2. BMBF will guide you through patching Beat Saber. Follow the on-screen steps: it often downloads required files and patches the Beat Saber install on-device.
  3. At times, BMBF may ask to uninstall or re-install Beat Saber temporarily — follow instructions carefully and keep any recommended backups.

5) Add custom songs & Quest-native mods using BMBF

  1. In BMBF you can manage songs: import songs from local files or from supported websites (BeatSaver, BeastSaber, etc.).
  2. Some Quest-native mods are distributed specifically for BMBF / Quest and can be added through BMBF's mod manager or by copying the mod files to the proper directory via SideQuest's file manager.
  3. After adding songs or mods, use the Sync to Beat Saber option in BMBF to apply changes. Then launch Beat Saber to test.

6) Maintain and update safely

  • When Beat Saber updates, your mod setup may break. Check BMBF and the mod authors for updated patches before updating Beat Saber.
  • Keep copies/backups of your custom songs and BMBF config so you can restore quickly if something goes wrong.

Troubleshooting

  • SideQuest doesn't detect headset: ensure Developer Mode is on, try a different USB cable/port, approve USB debugging in the headset.
  • BMBF patch failed: check log messages inside BMBF, reinstall BMBF via SideQuest, ensure Beat Saber version is supported.
  • Custom songs not showing: press "Sync to Beat Saber" in BMBF, then launch Beat Saber; confirm songs were imported and not corrupted.
  • Headset behaves oddly after patch: try rebooting the headset. If persistent, restore from backup or reinstall Beat Saber and re-run BMBF following official steps.

Best practices & tips

  • Always follow official docs for SideQuest and BMBF (links change over time).
  • Keep automatic Beat Saber updates disabled until you confirm mod compatibility with newer versions.
  • Make backups before patching: copy your Beat Saber install or use SideQuest's file manager to copy important files.
  • Join the BMBF / Quest modding community (Discord or forums) for the most up-to-date instructions and troubleshooting tips.
