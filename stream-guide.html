How to stream Beat Saber on Twitch using OBS (step-by-step)

  1. What you need first
    • PC with OBS Studio installed
    • Beat Saber on PC (Steam) or Quest using Link / Air Link
    • A Twitch account
    • A VR headset connected and working
  2. Connect OBS to Twitch
    1. Open OBS
    2. Go to Settings → Stream
    3. Set Service to Twitch
    4. Click Connect Account and log in (or use Stream Key if you prefer)
    5. Click Apply → OK
  3. Create a Beat Saber scene
    1. In Scenes, click + — Name it Beat Saber
    2. In Sources, click + and add:
      • Game Capture
        Mode: Capture specific window
        Window: Beat Saber
      • If that doesn’t work, try Window Capture
      • Use Display Capture only as a last resort
  4. Fix a black screen
    • Make sure OBS and Beat Saber are using the same GPU
    • Start Beat Saber before selecting it in OBS
    • Switch Game Capture → Window Capture if needed
    • Run OBS as administrator if nothing shows
  5. Set up audio

    Go to Settings → Audio and set:

    • Desktop Audio → your headset/speakers (game sound)
    • Mic/Aux → your microphone
    • Speak and check the audio bars are moving
  6. Video settings (recommended)

    Go to Settings → Video and set:

    • Base Resolution → 1920x1080
    • Output Resolution → 1280x720 or 1920x1080
    • FPS → 60
    • Click Apply
  7. Streaming quality settings

    Go to Settings → Output. Set Output Mode to Simple. Under Streaming:

    • Encoder → NVENC (if available)
    • Bitrate:
      • 720p60 → 4500 kbps
      • 1080p60 → 6000 kbps (Twitch max)
    • Click Apply
  8. Make Beat Saber look better (optional)
    • In Beat Saber Settings → Graphics: Enable Smooth Camera
    • Increase render scale if your PC can handle it
    • Resize/crop the game in OBS to remove black bars
  9. Go live
    1. Launch Beat Saber
    2. Check the game appears correctly in OBS
    3. Click Start Streaming
    4. Open your Twitch channel to confirm it’s live
  10. Quick fixes
    • Stream lagging → lower bitrate or resolution
    • No game audio → recheck Desktop Audio
    • Audio delay → Advanced Audio Properties → Sync Offset
    • Low FPS → close background apps or lower in-game settings
